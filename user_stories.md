# User Stories

## Doctor
- As a doctor, I want to view my appointments.
- As a doctor, I want to manage prescriptions.
- As a doctor, I want to view patient records.

## Patient
- As a patient, I want to book appointments.
- As a patient, I want to view my appointments.
- As a patient, I want to view prescriptions.

## Admin
- As an admin, I want to manage doctors.
- As an admin, I want to manage patients.
- As an admin, I want to generate reports.
